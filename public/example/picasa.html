Attributes

        

            
                

                    rule
                    Description
                    Default
                

            
            
                

                    thumb
                    是否开启缩略图模式
                    true
                

                

                    margin
                    接受数组，标记上右下左的外边距
                    [0,0,0,0]
                

                

                    padding
                    接受数组，标记上右下左内边距
                    [10, 50, 80, 50]
                

                

                    sideWidth
                    侧边栏宽度，设置则开启高级模式
                    0
                

                

                    skin
                    指定皮肤class样式
                    ""
                

                

                    data
                    数据，根据src、iframe依次检测是否是图片模式、iframe模式，最后是html模式
                    ""
                

                

                    setTitle(page,itm)
                    设置显示的标题
                    page/totalpage + itm.title
                

                

                    onchange(itm)
                    当切换的时候调用，在setTitle之后执行
                    
                

                

                    create
                    这个是UI方法的接口，UI初始化之后调用
                    
                

                

                    escExit
                    是否可以通过ESC按键关闭
                    true
