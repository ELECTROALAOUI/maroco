CSS规范 CSS

  1. .ac- 开头的类名不允许定义任何样式，含义是action，只用于js动作

  2. 图片处理 Images

    • 一般用png8(256色)、png24，photoshop导出的是png32
    • IE6不支持半透明 ，灰度图（小三角图）一般使用索引透明gif或png，请用xxx.8.png文件命名规范
    • 视情况而定把图片用css实现，减少http请求

JS规范 Javascript

  1. 一般界面模块使用UI({})来模块化
  2. 不允许创建全局变量
  3. 不要做容错处理，是bug就要暴露出来
  4. 对自身模块外的绑定比如$(window).resize(...)，必须提供自动或手动的销毁机制
  5. 如多个模块公用一个container，则模块需要自己生成一个innerContainer包裹。｛UI方法不设置el会自动生成一个el,在init中this.el.appendTo(container)｝
  6. IE67兼容注意
    • 用dom.innerHTML取代dom.text
